# 📒 Agenda Pessoal de Aniversários e Contatos

Este projeto consiste em uma **Agenda Pessoal** desenvolvida em Python, capaz de **cadastrar, procurar, atualizar, listar e excluir contatos**. Cada contato armazena informações como nome, aniversário, endereço, telefone, celular e e-mail.

## 📌 Objetivo

- Implementar as opções do menu com **validação dos inputs do usuário**.
- Utilizar, sempre que possível, as **funções já prontas no código base**.
- Trabalhar em **duplas** e entregar o projeto em **duas semanas** (não na próxima aula, mas na seguinte).

## 🧩 Funcionalidades

- 📇 **Cadastrar Contato**  
  Permite adicionar um novo contato, mantendo a lista ordenada alfabeticamente por nome. Utiliza **busca binária** para verificar se o nome já está cadastrado e determinar a posição correta de inserção.

- 🔍 **Procurar Contato**  
  Busca um contato pelo nome e exibe todas as suas informações. A busca pode ser cancelada a qualquer momento digitando `"cancela"`.

- ✏️ **Atualizar Contato**  
  Permite alterar os dados de um contato existente (exceto o nome). Um submenu permite escolher quais dados serão atualizados.

- 📃 **Listar Contatos**  
  Mostra todos os contatos cadastrados na agenda. Se não houver contatos, exibe uma mensagem informando.

- ❌ **Excluir Contato**  
  Exclui um contato existente após confirmação. Também permite cancelar a exclusão digitando `"cancela"`.

- 🚪 **Sair do Programa**  
  Encerra o programa com uma mensagem de sucesso.
